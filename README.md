# 🚲 Used Bike Price Prediction (ML Project)

## 📌 Overview
This project predicts the price of used bikes using Machine Learning models.

It uses:
- Linear Regression
- Decision Tree
- Random Forest

The project also compares model performance using evaluation metrics and cross-validation.

---

## 📊 Features
✔ Load dataset directly from URL (no Kaggle API)  
✔ Data preprocessing  
✔ Train multiple ML models  
✔ Model comparison (MAE, MSE, R2)  
✔ Cross-validation  
✔ Graph visualization  
✔ Feature importance  
✔ User input prediction  

---

- Python
- Pandas
